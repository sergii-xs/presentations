What is NFC?

Near Field Communication (NFC) is a short-range wireless technology that enables simple and secure communication between electronic devices.

  • Operates within a 4cm range (typically)
  • Based on RFID technology
  • Data transfer rates: 106-424 Kbit/s
  • Operates at 13.56 MHz frequency

How NFC Works

NFC functions through electromagnetic induction between two loop antennas:

Active Device

  • Generates RF field
  • Initiates communication
  • Powered (e.g., smartphones)

Passive Device

  • Powered by the RF field
  • No battery required
  • Responds to requests

NFC Communication Modes

  • Reader/Writer: Active device reads/writes to passive NFC tags
  • Peer-to-Peer: Two active devices exchange data
  • Card Emulation: Active device mimics a passive NFC card

NFC Tags Explained

NFC tags are small, passive devices containing:

  • An integrated circuit (chip)
  • An antenna
  • Optional memory (varies by type)

They can be programmed to perform specific actions when scanned.

Types of NFC Tags

Type Memory Speed Common Uses
Type 1 96 bytes - 2KB 106 Kbit/s Simple tasks, URL sharing
Type 2 48 bytes - 2KB 106 Kbit/s Business cards, simple links
Type 3 (FeliCa) Up to 1MB 212 Kbit/s Transit cards, payments
Type 4 Up to 32KB 424 Kbit/s Access control, advanced applications
Type 5 (MIFARE) Up to 8KB Varies Transit, access control

Form Factors

  • Stickers: Thin, adhesive-backed tags for surfaces
  • Cards: Credit card sized for wallets and holders
  • Key fobs: Durable tags for keychains
  • Wristbands: Wearable NFC for events and fitness
  • Implantables: Biocompatible tags for under-skin use
  • Custom shapes: Embedded in products or promotional items

What Can NFC Tags Do?

  • Open a website or app
  • Connect to Wi-Fi networks
  • Share contact information
  • Trigger phone settings (Airplane mode, Bluetooth, etc.)
  • Launch navigation to a location
  • Make a phone call or send a text
  • Trigger home automation
  • Check-in on social media

What to Look For When Buying

  • Memory size: Based on your use case needs
  • Form factor: Stickers, cards, wristbands, etc.
  • NFC Type: Type 1-5 (NTAG213/215/216 are common)
  • Durability: Weather-resistant for outdoor use
  • Price: Typically $0.30-$2 per tag (bulk discounts available)
  • Compatibility: Ensure they work with your devices

Recommended NFC Tag Types

  • General use: NTAG213 (144 bytes)
  • More data: NTAG215 (504 bytes)
  • Advanced projects: NTAG216 (888 bytes)
  • Amiibo cloning: NTAG215 specifically
  • Outdoor/durable: Look for IP-rated tags
